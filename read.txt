
    1. Структура: Код разделен на несколько классов для лучшей организации. Main отвечает за взаимодействие с пользователем, CaesarCipher реализует логику шифрования и расшифровки, FileHandler управляет чтением и записью файлов, а Validator проверяет корректность входных данных.
    2. Шифрование и расшифровка: Реализованы методы для шифрования и расшифровки текста с использованием заданного сдвига. Также предусмотрен метод для перебора всех возможных сдвигов.
    3. Валидация: Программа проверяет существование файлов и корректность сдвига перед выполнением операций.
    4. Выход из программы: Программа завершает работу при вводе "exit".
